ROT20

The ROT20 cipher is a variation of the caesar cipher, which is a simple form of substitution cipher.
The caesar cipher was one of the very earliest known forms of encryption and involved taking each letter in a message then rotating them by a number of places in the alphabet.
For example, "A" would become "B", "B" would become "C" and "Z" would become "A".
In ROT20, each letter is rotated by twenty places, so "A" becomes "U", "B" becomes "V" and "Z" comes "T".

Reverse

The reverse cipher not only reverses the order of the user input, but also reverses the order of each word.
This method of encryption can be very effective if the person trying to decrypt the message assumes a substitution cipher has been used when the solution is "hiding in plain sight".
Unfortuantely, this method of cipher is not as effective if you include a palindrome in the user input.

ROT20Reverse

This cipher combines both the ROT20 and Reverse ciphers to form a combo-cipher that reverses the order of words and letters from the user input then rotates the letters by 20 places.
Combining ciphers is a very good way to strengthen encryption. If one of the combined ciphers is broken, there is still another layer to be deciphered.
